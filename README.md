# Bus Pass Management System

## Overview
The Bus Pass Management System is a Java Swing desktop application developed to automate the process of issuing and managing bus passes. It provides separate interfaces for users and administrators, reducing paperwork and simplifying pass management.

## Features

- User Registration
- User Login
- Admin Login
- Apply for New Bus Pass
- View Bus Passes
- Admin Dashboard
- Pass Approval and Management
- MySQL Database Integration

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans IDE
